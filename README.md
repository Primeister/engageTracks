# EngageTrack

A web-based engagement tracking system built with **HTML, CSS, JavaScript, PHP, and MySQL**. The application helps organizations monitor attendance and engagement by providing an intuitive platform for managing participants, recording attendance, and generating insightful reports.

## 🌐 Overview

EngageTrack is designed to simplify engagement management by replacing manual record-keeping with a centralized web application. It enables administrators to manage participants and events while providing accurate attendance records and engagement statistics.

## ✨ Features

### 👤 User Features
- Register and log in securely
- View scheduled events or sessions
- Track attendance and engagement history
- Update personal profile information

### 👨‍💼 Administrator Features
- Manage users and participant records
- Create and manage events or engagement sessions
- Record and update attendance
- View engagement statistics and reports
- Search and filter records
- Generate attendance summaries

## 🛠️ Built With

- **CSS3**

## 🚀 Installation

1. Clone the repository.

2. Place the project inside your web server directory (e.g., `htdocs` for XAMPP).

3. Import the provided SQL database into MySQL.

4. Update the database connection settings in the configuration file.

5. Start Apache and MySQL.

6. Open the application in your browser.

```
http://localhost/engagetrack
```

## 📖 How to Use

1. Log in using your account.
2. Navigate to the dashboard.
3. Create or manage engagement sessions.
4. Record participant attendance.
